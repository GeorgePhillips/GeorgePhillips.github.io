Welcome

We’re really excited to have you using Jekyll on CloudCannon. There are a few things you should know.

Initial Setup

For a typical CloudCannon website you just need to create the site and add the files. For jekyll sites this is no different. Just add your jekyll files and make sure there is a config.yml file present.

CloudCannon will sync the files and compile it with every change.

Blogging Interface

One of the new things with the private beta is the blogging interface. CloudCannon will automatically make sense of your directory structure. With this CloudCannon gives you a list of your blog posts and drafts, allows you to create new one and even write your content in a visual way rather than using markdown. Of course you can still update the code directly as you usually do.

Editing Post Headers

Editing the post headers is usually just a matter of editing some yaml at the top of your post. To do this visually we need some more information. Introducing /_posts/_defaults.json file. This defines what the default headers are for new posts and how it can be changed. Let take a look at CloudCannon's /_posts/_defaults.json file:

{
  "layout": "post",
  "title": "Enter a better title here!",
  "header": "",
  "category": ["Features", "Freebies", "Company", "Tutorials", "Interview"],
  "author": ["george", "mike", "yasmin"]
}

CloudCannon reads these values and uses them to generate the header for any new post. A new post will get the following header: 

---
layout: post
title: Enter a better title here!
header: 
category: Features
author: george
---

When this post is opened in the visual editor a cog will appear in the top left corner. Clicking this will open a settings dialog.

Within this dialog the /_posts/_defaults.json file is used again and the lists (category, author) are turned into dropdown menus. As well as this the contents of the _layouts directory are read to give a list of layouts. With this you can configure how much configuration your clients can have.

Plugins

Plugins are not run on CloudCannon for security reasons. If there is one that you find essential for all projects please contact us.
