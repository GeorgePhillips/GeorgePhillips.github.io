FAQ

Q: How is this different from Dropbox Public Folders?
A: Using Dropbox Public Folders is an easy way to share files on the internet, CloudCannon is a platform for web designers so our goals are slightly different. CloudCannon provides an integrated content management system and a host of other features to make a designer's life easier. We also have no bandwidth restrictions for sites hosted on CloudCannon and do a number of optimizations to make the sites load as fast as possible.
Q: Can I use SQL/PHP/Rails with CloudCannon?
A: No, CloudCannon is focussed on front-end technology. However, CloudCannon provides powerful templating features to replace these technologies (See our docs for more information). You can also add dynamic content by embedding 3rd party services such as Wufoo or Disqus.
Q: What file types are supported?
A: We currently sync files with the following extensions - js, css, less, gif, jpe, jpg, jpeg, png, htm, html, pdf, txt, zip, swf, svg, json, xml, woff, eot, ttf, tte, otf, ico, htc, xsl, mp3, m4a, ogg, oga, web, wav, mp4, web, ogv, map.
Q: Can I use my own domain?
A: Yes. You can either use your own domain or we supply <yourname>.cloudvent.net domains.
Q: Are there any file size limits
A: Yes, there's a 200MB limit on files synced through Dropbox or the website. Anything bigger probably belongs on YouTube or a 3rd party file sharing website.
Q: Do you provide email hosting?
A: No, however you can easily use an emailing hosting specialist alongside CloudCannon. Here's a list of the best email hosts out there.
Q: Does CloudCannon support SAML based SSO?
A: Yes, and our partner OneLogin has a free plan for CloudCannon - find out more at our OneLogin partner page. To set this up email us at support@cloudcannon.com.
Q: What elements can be made editable?

A: We split our editable elements into three different categories; Block, text and special elements. Each type acts slightly different in the editor.

  • Block elements: div, section, article, aside, footer, header, nav
  • Text elements: p, h1, h2, h3, h4, h5, h6
  • Special elements: img
Q: How can I get in touch?
A: We'd love to hear from you. Email us any time at support@cloudcannon.com.
